# Pneumonia-Detection-AI

Chest X-ray classification system for detecting pneumonia using Deep Learning and Transfer Learning models.

## Project Overview

This project aims to detect pneumonia from chest X-ray images using multiple deep learning architectures.

## Models Used

### 1. Custom CNN
- Test Accuracy: 89.74%
- Test Loss: 0.2543

### 2. MobileNetV2
- Test Accuracy: 87.98%
- Test Loss: 0.2988

### 3. ResNet50 (Best Model)
- Test Accuracy: 91.99%
- Test Loss: 0.2148

## Model Comparison

| Model | Test Accuracy | Test Loss |
|-------|--------------|-----------|
| CNN | 89.74% | 0.2543 |
| MobileNetV2 | 87.98% | 0.2988 |
| ResNet50 | 91.99% | 0.2148 |

## Best Model

ResNet50 achieved the best performance.
